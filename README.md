# Teste Prático React – COVID-19

Aplicação React desenvolvida como teste técnico, consumindo a API pública
covid19-brazil-api para exibição de dados da COVID-19 no Brasil e no mundo.

## Tecnologias
- React
- TypeScript
- Tailwind CSS
- Axios

## Funcionalidades
- Consulta do status atual da COVID-19 em todos os estados do Brasil
- Consulta do status da COVID-19 por estado selecionado
- Consulta do status da COVID-19 no Brasil por data
- Consulta do status da COVID-19 em outros países
- Formulário com validação para simulação de envio de dados
